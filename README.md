# CyLab-Writeups-challenges

My writeup on the [CyLab Security Academy](https://learn.cylabacademy.org/) challenges I solved.


This repository show and document how i solved each different Lab.

## Structure

All Writeups are classified by Difficulty in their category :  
- `01-General-Skills`
- `02-Web-Exploitation`
- `03-Cryptography`
- `04-Blockchain`
- `05-Reverse-Engineering`
- `06-Binary-Exploitation`
- `07-Forensics`


## Writeup format

Each writeup includes:
- The Challs context
- The vulnerability explained briefly
- Step-by-step solution
- Key takeaway
